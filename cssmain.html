CSS 1페이지다.

안녕하신가! 힘세고 강한 아침, 만일 내게 물어본다면 나는 css페이지.


CSS는 HTML문서를 시각적으로 꾸미는 역할을 한다. 일괄적으로 적용할수 있으며 외부에서 적용하는것이기 때문에 보다 간편하게 적용할수 있다. 이때 필요한 필수명칭을 잠깐 설명하겠다.

요소 설명 실행
< 선택자 >
CSS는 외부 스타일시트이다. 따라서 HTML에 스타일을 적용할때 요소에 접근해줘야한다. 이때 사용하는것이 선택자이다.
너도 교수님이 이름을 불러야 대답하지 않나?
< 속성 > 선택자로 대상을 지정해줬으면 본격적으로 스타일을 지정해줘야겠지. 이때 속성을 줘서 스타일을 변경할수 있다. 예를들어 color 속성을 통해 색상을 변경하거나 font-size로 폰트 크기를 변경할수있다.
< 속성값 > 속성을 정해줬다면 속성값도 줘야겠지. 앞서 설명한 color 속성에 속성값으로 blue로 정의하고 font-size를 5px로 정의하면 5픽셀짜리 퍼런 글씨가 나온다!
< 선언 > 선언은 속성과 속성값을 결합하여 스타일을 지정하는 부분이다. 속성도 속성값도 선언의 중괄호('{}')안에서 아늑하게 지내겠지. 각각 세미콜론으로 구분한다. 끝에 세미콜론을 빼먹는 실수는 하지마라.

요소

버튼을 눌러 생김새를 보자. 간단한 예시다.
html에서 p를 선택해 색을 빨간색으로 폰트 크기를 9px로 정의하는 css다. p { color: red ; pont-size: 9px ; }
